# Telecom Customer Churn Analysis (Excel)

## Project Objective
To analyze customer churn patterns in the telecom industry and identify
key factors influencing customer retention.

## Tools & Skills Used
- Microsoft Excel
- Data Cleaning
- Pivot Tables
- Charts & Dashboard
- Exploratory Data Analysis (EDA)

## Dataset Description
The dataset includes customer demographics, service usage, contract type,
payment methods, and churn status.

## Key Insights
- Month-to-month contract customers show higher churn
- Customers using electronic check payment have higher churn rate
- Long-term contract customers are more loyal

## Dashboard
Dashboard screenshots are included in the Screenshots folder.

## Conclusion
This project helps understand customer behavior and provides insights
to reduce customer churn in telecom companies.
